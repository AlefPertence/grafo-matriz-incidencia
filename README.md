Grafo - Matriz de Incidência

Programa em C que representa um grafo usando matriz de incidência (vértices nas linhas, arestas nas colunas).

O que o programa faz


Insere arestas entre vértices
Mostra a matriz de incidência
Calcula o grau de um vértice ou de todos
Lista as arestas do grafo
Limpa o grafo
